#head3improve_2{color: black;}

Selector : ID Selector

Why ? : The selector starts with a '#' symbol, indicating it's selecting an element by its unique ID, and in this case, it's selecting the element with the ID 'head3improve_2'.

.moviestitle {background-color: #4caf4f2a;}

Type : Class Selector

Why ? : The selector starts with a '.', indicating it's selecting elements by class. It's applied to elements with the class 'moviestitle', and classes can be reused on multiple elements.

p {text-indent: 50px;text-align: justify;}

Type : Tag Selector

Why ? : The selector is just the tag name 'p', indicating it's selecting all 'p' (paragraph) elements. This is a type selector, as it selects elements based on their tag name.

.highlightbook {font-weight: bold;color: white;}

Type : Class Selector

Why ? : Similar to the second example, it uses a class selector indicated by '.', applying styles to all elements with the class 'highlightbook'.

#stanlee{letter-spacing: 5px;color: blue;}

Type : ID Selector

Why ? : Similar to the first example, it's using an ID selector ('#stanlee') to select a specific element by its unique ID, applying styles only to that element.

h3 {text-align: left;color: #4CAF50;}

Type : Tag Selector

Why ? : The selector is the tag name 'h3', selecting all 'h3' elements. It's a type selector as it selects elements based on their tag name.
